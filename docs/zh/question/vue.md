# 打包 vue/react 项目

将 vue/react 项目进行 build 之后，将 dist 文件夹中的文件上传到服务器即可。

## 注意事项

-   上传的文件夹必须包含 index.html 文件。
-   上传的文件夹中单个文件的大小不能超过 10MB。
-   静态资源最好使用外链的形式，例如图片、视频等。

<script setup>
import BiliBili from "../components/bilibili.vue"
</script>
